Class Hierarchy

  • exceptions.BaseException: Common base class for all exceptions

      • googlevoice.util.ForwardingError: Forwarding number given was incorrect
      • googlevoice.util.JSONError: Failed JSON deserialization
      • googlevoice.util.LoginError: Occurs when login credentials are incorrect
      • googlevoice.util.ParsingError: Happens when XML feed parsing fails
      • exceptions.StandardError: Base class for all standard Python exceptions that do not represent interpreter exiting.
        • exceptions.AssertionError: Assertion failed.
      • googlevoice.util.ValidationError: Bombs when response code back from Voice 500s

    • googlevoice.tests.VoiceTest
  • googlevoice.voice.Voice: Main voice instance for interacting with the Google Voice service Also contains callable methods for each folder (eg inbox,voicemail,sms,etc)
  • dict: dict() -> new empty dictionary.

      • googlevoice.voice.Message: Wrapper for all call/sms message instances stored in Google Voice...
      • googlevoice.voice.Phone: Wrapper for phone objects used for phone specific methods...
    • googlevoice.util.XMLParser: XML Parser helper that can dig json and html out of the feeds Calling the parser returns a tuple of (data_dict, html_content)
